### Stack Tecnológico

# Control de versiones
- `Git` Sistema de control de versiones ampliamente usado que permite hacer seguimiento a ficheros y coordinar el trabajo entre distintas personas.
- `Github` Plataforma que puede mantener repositorios de código en almacenamiento basado en la nube y que facilita el uso de `Git`.

# Backend
- `Django Web Framework` Framework web muy usado para programar con lenguaje `Python` y *open-source* que acelera el desarrollo.

# Frontend
- `HTML`, `CSS`, `JavaScript` Lenguajes básicos e imprescindibles para definir la estructura, presentación y comportamiento de los elementos de cualquier sitio web.
-  `Boostrap Framework` Uno de los más frameworks `CSS` más populares desarrollado por *Twitter* que facilita muchos aspectos del diseño web

# Base de datos
- `MySQL` Uno de los más populares sistemas de gestión de bases de datos relacionales con gran popularidad para acceder, añadir y manejar una base de datos.

# Despliegue
- `Docker` Plataforma que facilita el despliegue, desarrollo y funcionamiento de aplicaciones a través del uso de contenedores ligeros y portables que se ejecuten en cualquier maquina independientemente del sistema operativo.
